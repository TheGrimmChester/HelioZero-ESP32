Third-party install packs referenced from this repo (not embedded in ESP32 flash).

| Integration | Install | Documentation |
|-------------|---------|---------------|
| Home Assistant — HACS custom component (REST polling) | [HelioZero-HACS](https://github.com/TheGrimmChester/HelioZero-HACS) | [HACS pack](https://heliozero.clouded.fr/en/integrations/home-assistant/hacs/) |

Automations, Lovelace, and Energy dashboard examples are on [heliozero.clouded.fr](https://heliozero.clouded.fr/en/integrations/home-assistant/) (source: [HelioZero-Website](https://github.com/TheGrimmChester/HelioZero-Website) under `content/*/integrations/home-assistant/` and `assets/integrations/home-assistant/`).
